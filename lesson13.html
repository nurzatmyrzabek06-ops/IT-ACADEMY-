Видеоны көру үшін басыңыз

1️⃣ while циклін қашан қолданамыз? Теория
🔄 while циклі дегеніміз не? Негіз
while циклі — белгілі бір шарт ақиқат (True) болып тұрғанша, блок ішіндегі кодты қайталап орындайды. Шарт жалған (False) болған кезде цикл тоқтайды.

Синтаксис:
while шарт: орындалатын код блогы ... # шарт жалған болғанда цикл тоқтайды
📌 while циклін қашан қолданамыз? — Қанша рет қайталану керек екені алдын ала белгісіз болғанда
— Пайдаланушы дұрыс деректер енгізгенше қайта сұрау қажет болғанда
— Ойын, мәзір немесе шарт орындалмайынша тоқтамайтын бағдарлама жазғанда
— Сервер, датчик немесе файлдан деректер кіргенше күту керек болғанда
✅ while циклі қолдан
while пайд_жауап != "иә":
    жауап = input("Шыға аласың ба?")

while санақ > 0:
    print(санақ)
    санақ -= 1
🔁 for циклі қолдан
for i in range(5):
    print(i)

for зат in тізім:
    print(зат)
🧠 Негізгі айырмашылық: for — санын білгенде · while — шартқа байланысты болғанда
2️⃣ Қауіпті шексіз цикл Ескерту
⚠️ Шексіз цикл (Infinite Loop) Қауіп
Егер цикл шарты ешқашан жалған болмаса, цикл мәңгіге орындала береді. Бұл бағдарламаның тоңып қалуына немесе бүкіл ресурсты жеп қоюына алып келеді.
⛔ Шексіз цикл мысалдары — ҚАТЕ КОД
1-мысал — санауышты ұмыту:
i = 0
while i < 5:
    print(i)  ← i += 1 жазылмаған!

2-мысал — while True:
while True:
    print("Сәлем")  ← break жоқ, тоқтамайды!

3-мысал — шарт ешқашан өзгермейді:
x = 10
while x > 0:
    print("жұмыс")  ← x азаймайды!
🛡️ Шексіз циклден қалай қорғанамыз? 1. Санауышты міндетті түрде өзгерт: i += 1 немесе i -= 1
2. break операторын шарт ішінде қолдан
3. while True қолдансаң — break жазуды ұмытпа
4. Максималды қайталану санауышын қос: if кезек > 100: break
while True: # шексіз цикл жауап = input("Шығу үшін 'q' жаз: ") if жауап == "q": break # ✅ break арқылы тоқтатамыз
3️⃣ Практикалық мысал Теория
💡 Нақты өмірдегі мысалдар Мысал
🔐 Мысал 1 — Парольді тексеру Дұрыс пароль енгізілмейінше қайта сұрайды:
пароль = "" while пароль != "1234": пароль = input("Парольді жаз: ") if пароль != "1234": print("❌ Қате пароль, қайтадан!") print("✅ Сәлем, кіруіңіз рұқсат!")
🔢 Мысал 2 — Кері санақ таймері 10-нан 1-ге дейін санайды:
санақ = 10 while санақ > 0: print(f"🕐 {санақ}...") санақ -= 1 print("🚀 Ұшуға дайын!")
🎲 Мысал 3 — Жасырын санды табу ойыны Пайдаланушы дұрыс санды тапқанша сұрайды:
жасырын = 42 болжам = 0 while болжам != жасырын: болжам = int(input("Санды болжа (1-100): ")) if болжам < жасырын: print("📈 Көбірек!") elif болжам > жасырын: print("📉 Азырақ!") print("🎉 Дұрыс таптың!")
🧪 Практикалық есептер 3 есеп
🔢 Есеп 1: Натурал сандардың қосындысы
📋 Тапсырма мазмұны: Пайдаланушыдан бір оң бүтін n санын сұра. while циклін қолданып, 1-ден n-ге дейінгі барлық натурал сандардың қосындысын есепте және экранға шығар.

Мысалы: n = 5 болса → 1+2+3+4+5 = 15

Қолдан: while · int(input()) · +=
task1.py
1
Шығарылу жолы:
n = int(input("n санын жаз: "))
i = 1
қосынды = 0
while i <= n:
    қосынды += i
    i += 1
print("Қосынды:", қосынды)
Нәтиже осы жерде шығады...
🔐 Есеп 2: Пароль тексергіш
📋 Тапсырма мазмұны: Дұрыс пароль: "python123". Пайдаланушыдан пароль сұра. Егер қате болса, "Қате пароль! Қайталаңыз." деп, қайта сұрай бер. Дұрыс пароль енгізілгенде "✅ Қош келдіңіз!" деп шығар.

Қолдан: while · != · input()
task2.py
1
Шығарылу жолы:
пароль = ""
while пароль != "python123":
    пароль = input("Парольді жаз: ")
    if пароль != "python123":
        print("Қате пароль! Қайталаңыз.")
print("✅ Қош келдіңіз!")
Нәтиже осы жерде шығады...
🎲 Есеп 3: Жұп сандарды жинақтау
📋 Тапсырма мазмұны: Пайдаланушыдан n санын сұра. while циклін пайдаланып, 1-ден n-ге дейін тек жұп сандарды экранға жолыма шығар, соңында олардың санын да жаз.

Мысалы: n = 10 → 2 4 6 8 10, Жұп сандар саны: 5

Қолдан: while · % (модуль) · print(..., end=" ")
task3.py
1
Шығарылу жолы:
n = int(input("n санын жаз: "))
i = 1
сан = 0
while i <= n:
    if i % 2 == 0:
        print(i, end=" ")
        сан += 1
    i += 1
print()
print("Жұп сандар саны:", сан)
Нәтиже осы жерде шығады...
