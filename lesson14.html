1️⃣ list деген не? Теория
📋 Тізім (list) — Python-дың негізгі деректер құрылымы Негіз
List (тізім) — бірнеше мәнді бір айнымалыда сақтауға мүмкіндік беретін деректер құрылымы. Тізімдегі элементтер реттелген, өзгертілетін және қайталанатын болуы мүмкін.

Тізім квадрат жақша [ ] ішінде жазылады, элементтер үтір арқылы бөлінеді.
# Бос тізім тізім = [] # Сандар тізімі сандар = [1, 2, 3, 4, 5] # Жолдар тізімі атаулар = ["Аяша", "Берік", "Дина"] # Аралас тізім аралас = [42, "python", 3.14, True] # Тізімнің ұзындығы print(len(сандар)) # → 5
🧠 list не үшін керек? — Оқушылардың бағаларын бір жерде сақтауға
— Дүкен тізімін, контактілерді жинауға
— Бірнеше нәтижені есептеп, сақтауға
— Циклмен топтық деректерді өңдеуге
✅ list негізгі қасиеттері: реттелген — элементтердің орны бекітілген
өзгертілетін — элемент қосуға, өшіруге, ауыстыруға болады
қайталанатын — бір мән бірнеше рет болуы мүмкін
аралас типтер — int, str, float, bool бір тізімде тұра алады
2️⃣ Индекс Теория
🔢 Индекстеу — элементке жету жолы Маңызды
Тізімдегі әр элементтің өз индексі (нөмірі) бар. Python-да индекс 0-ден басталады. Сонымен қатар теріс индекс арқылы соңынан санауға болады.

жемістер = ["алма", "алмұрт", "өрік", "жүзім", "банан"] # Оң индекс — басынан print(жемістер[0]) # → алма print(жемістер[2]) # → өрік # Теріс индекс — соңынан print(жемістер[-1]) # → банан print(жемістер[-2]) # → жүзім # Элементті ауыстыру жемістер[1] = "қарбыз" print(жемістер[1]) # → қарбыз # Тілім алу (slicing) print(жемістер[1:4]) # → ['қарбыз', 'өрік', 'жүзім']
⚠️ IndexError — жиі кездесетін қате: Тізімде 5 элемент болса, индекс 0–4 аралығында ғана. тізім[5] жазса — IndexError шығады!
3️⃣ Қосу / өшіру Теория
🛠️ Тізімді өзгерту әдістері Методтар
Әдіс Не істейді Мысал Нәтиже
.append(x) Соңына қосады т.append("жаңа") [..., "жаңа"]
.insert(i, x) i орынға қосады т.insert(1, "орта") [т[0], "орта", ...]
.remove(x) Бірінші x өшіреді т.remove("алма") "алма" жоқ
.pop() Соңғы өшіреді + қайтарады т.pop() соңғы элемент
.pop(i) i-ші өшіреді + қайтарады т.pop(0) бірінші элемент
.sort() Өсу ретімен сорттайды т.sort() [1, 2, 3, ...]
.reverse() Тізімді керіге айналдырады т.reverse() [соңы, ..., басы]
len(т) Элементтер санын береді len([1,2,3]) 3
азық = ["нан", "сүт", "жұмыртқа"] # Соңына қосу азық.append("май") print(азық) # → ['нан', 'сүт', 'жұмыртқа', 'май'] # Белгілі орынға қосу азық.insert(1, "қант") print(азық) # → ['нан', 'қант', 'сүт', 'жұмыртқа', 'май'] # Мән арқылы өшіру азық.remove("сүт") print(азық) # → ['нан', 'қант', 'жұмыртқа', 'май'] # Соңғы элементті шығарып өшіру шыққан = азық.pop() print(шыққан) # → май print(азық) # → ['нан', 'қант', 'жұмыртқа']
4️⃣ Циклмен жұмыс Теория
🔄 for циклімен тізімді өңдеу Практика
for циклі тізімнің әр элементін кезекпен алып өңдейді. Бұл — тізімдермен жұмыстың ең жиі қолданылатын тәсілі.
# Мысал 1 — элементтерді шығару жануарлар = ["мысық", "ит", "қоян"] for жануар in жануарлар: print(жануар) # → мысық # → ит # → қоян # Мысал 2 — индекспен бірге (enumerate) for i, зат in enumerate(жануарлар): print(i, зат) # → 0 мысық # → 1 ит # → 2 қоян # Мысал 3 — сандардың қосындысы бағалар = [85, 92, 78, 96, 88] қосынды = 0 for баға in бағалар: қосынды += баға print(f"Орташа баға: {қосынды / len(бағалар)}") # → Орташа баға: 87.8 # Мысал 4 — шарт бойынша сүзу for баға in бағалар: if баға >= 90: print(баға, "— Үздік!") # → 92 — Үздік! # → 96 — Үздік!
💡 in операторы — тізімде бар-жоғын тексеру: "алма" in жемістер → True немесе False қайтарады
if "алма" in жемістер: print("Бар!")

Нәтиже үлгісі: Қосынды: 410 және Орташа: 82.0

Қолдан: list · for ... in · len() · +=
