🧪 Практикалық есептер 2 есеп
🌡️ Есеп 1: Ауа райы болжамы
📋 Тапсырма мазмұны: Пайдаланушыдан ауаның температурасын (°C) сұра. Температураға байланысты кеңес бер:

— 35°C-тан жоғары → "Өте ыстық! Үйде қал."
— 20–35°C аралығы → "Жылы, серуендеуге болады."
— 5–19°C аралығы → "Салқын, жейде ки."
— 5°C-тан төмен → "Суық! Пальто ки."

Қолдан: if · elif · else · and
task1.py
1
Шығарылу жолы:
temp = int(input("Температураны жаз (°C): "))
if temp > 35:
    print("Өте ыстық! Үйде қал.")
elif temp >= 20 and temp <= 35:
    print("Жылы, серуендеуге болады.")
elif temp >= 5 and temp <= 19:
    print("Салқын, жейде ки.")
else:
    print("Суық! Пальто ки.")
Нәтиже осы жерде шығады...
🎓 Есеп 2: Баға жүйесі
📋 Тапсырма мазмұны: Пайдаланушыдан тест нәтижесін (0–100 балл) сұра. Баллға сәйкес баға қой:

— 90–100 → "5 — Өте жақсы"
— 75–89 → "4 — Жақсы"
— 55–74 → "3 — Қанағаттанарлық"
— 0–54 → "2 — Қанағаттанарлықсыз"
— Диапазоннан тыс (теріс немесе >100) → "Қате енгізу! 0–100 аралығында болуы керек."

Қолдан: if · elif · else · and · or
task2.py
1
Шығарылу жолы:
score = int(input("Балыңды жаз (0-100): "))
if score < 0 or score > 100:
    print("Қате енгізу! 0–100 аралығында болуы керек.")
elif score >= 90:
    print("5 — Өте жақсы")
elif score >= 75:
    print("4 — Жақсы")
elif score >= 55:
    print("3 — Қанағаттанарлық")
else:
    print("2 — Қанағаттанарлықсыз")
