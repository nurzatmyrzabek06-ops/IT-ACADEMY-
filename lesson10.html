🔢 Тапсырма 1: Жасын тексер
📋 Тапсырма мазмұны: Пайдаланушыдан жасын сұра. Егер жасы 18-ден үлкен немесе тең болса — «Кәмелетке толған» деп шығар, болмаса — «Кәмелетке толмаған» деп шығар.

Қолдан: age = int(input(...)) · if age >= 18: · else: · print(...)
task1.py
1
Шығарылу жолы:
age = int(input("Жасыңды жаз: "))
if age >= 18:
    print("Кәмелетке толған")
else:
    print("Кәмелетке толмаған")

⚖️ Тапсырма 2: Екі санды салыстыр
📋 Тапсырма мазмұны: Пайдаланушыдан екі сан алып, оларды салыстыр.
— Бірінші үлкен болса → «Бірінші сан үлкен»
— Екінші үлкен болса → «Екінші сан үлкен»
— Тең болса → «Сандар тең»

Қолдан: a > b · a < b · a == b · elif
task2.py
1
Шығарылу жолы:
a = int(input("Бірінші санды жаз: "))
b = int(input("Екінші санды жаз: "))
if a > b:
    print("Бірінші сан үлкен")
elif a < b:
    print("Екінші сан үлкен")
else:
    print("Сандар тең")
Нәтиже осы жерде шығады...
🎟️ Тапсырма 3: Билет бағасы
📋 Тапсырма мазмұны: Пайдаланушының жасы мен студент екендігін сұра.
— Жасы 7-ден кіші болса → «Тегін»
— Жасы 7–17 аралығында НЕМЕСЕ студент болса → «Жеңілдік: 500 теңге»
— Басқа жағдайда → «Толық баға: 1000 теңге»

Қолдан: and · or · not · elif
task3.py
1
Шығарылу жолы:
age = int(input("Жасыңды жаз: "))
student = input("Студент пе? (иә/жоқ): ")
is_student = student == "иә"
if age < 7:
    print("Тегін")
elif (age >= 7 and age <= 17) or is_student:
    print("Жеңілдік: 500 теңге")
else:
    print("Толық баға: 1000 теңге")

📊 Тапсырма 5: Үш санның ішіндегі ең үлкені
📋 Тапсырма мазмұны: Пайдаланушыдан үш сан алып, ең үлкенін тап.
— a ең үлкен болса → «Ең үлкен сан: [a]»
— b ең үлкен болса → «Ең үлкен сан: [b]»
— c ең үлкен болса → «Ең үлкен сан: [c]»

Қолдан: a >= b and a >= c · elif · f"..."
task5.py
1
Шығарылу жолы:
a = int(input("Бірінші сан: "))
b = int(input("Екінші сан: "))
c = int(input("Үшінші сан: "))
if a >= b and a >= c:
    print(f"Ең үлкен сан: {a}")
elif b >= a and b >= c:
    print(f"Ең үлкен сан: {b}")
else:
    print(f"Ең үлкен сан: {c}")
